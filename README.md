# linux-eye
A Linux security monitoring tool that watches your system for suspicious activity in real-time.

## what does it monitor
This tool monitor all running processes continuously and give warning if any suspicious process found.

## How do you install it
To install this tool just clone the repo by typing on terminal 

git clone https://github.com/amitpadhan525/linux-eye.git

After running this install requirments 

pip install -r requirments.txt

After that start the tool

python -m linux_eye.main.py
